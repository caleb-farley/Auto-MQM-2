Frequently Asked Questions

What is MQM and how does this tool use it?

MQM (Multidimensional Quality Metrics) is an industry-standard framework for evaluating translation quality. This tool automatically detects issues like mistranslations, fluency problems, and terminology errors using MQM categories.

How accurate is the assessment?

The tool uses a combination of LLMs and linguistic rules to provide reliable assessments. While it's not a substitute for a human reviewer in critical cases, it's highly effective for scalable QA and triage.

Does this integrate with my CAT tool or TMS?

Not yet, but we’re actively building integrations. Let us know which tools you’d like us to support first.

Can I use this for technical content like regulatory or medical content?

Yes, the tool supports high-stakes content. However, we recommend using it as a pre-QA step before final human validation for regulatory texts.

How do you handle data privacy?

Your data is never stored or reused. We process inputs securely and ensure that no text is saved after the session ends.
